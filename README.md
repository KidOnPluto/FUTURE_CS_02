# FUTURE_CS_02 – Phishing Email Detection & Awareness

## Author

Oratile Maretela

## Task Overview

This project focuses on identifying and analyzing phishing emails and educating users on how to avoid them.

## Contents

* Phishing Report
* Screenshots of email examples

## Key Findings

* Phishing emails create urgency
* Fake domains are common
* Users are tricked into clicking malicious links

## Prevention

* Verify sender emails
* Avoid suspicious links
* Use 2FA

##  Conclusion

Awareness is the best defense against phishing attacks.
